## Features

- **Dark / Light Mode** — Theme toggle with localStorage persistence  
- **Smooth Scroll Navigation** — Anchor-based navigation with active link tracking  
- **Project Showcase** — Cards with video modal previews and GitHub links  
- **Certificate Carousel** — Responsive Swiper slider (2–4 slides per view)  
- **Contact Form** — Sends real emails via EmailJS  
- **CV Download** — One-click PDF download  
- **Fully Responsive** — Mobile-first design with custom breakpoints  
- **Performance** — Lazy-loaded routes and images, Vite code splitting
